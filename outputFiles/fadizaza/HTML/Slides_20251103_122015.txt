!!! Introduction to HTML: Building Blocks of the Web

*   **What is HTML?** HTML stands for HyperText Markup Language. Think of it as the skeleton of a webpage – it provides the structure and content. It's the language your browser (like Chrome or Firefox) uses to understand what to display.
*   **HTML uses Tags:** These are special keywords surrounded by angle brackets, like `<p>` for paragraph or `<h1>` for a main heading. Most tags come in pairs: an opening tag (`<p>`) and a closing tag (`</p>`). The closing tag has a forward slash.
*   **Basic HTML Structure:** Every HTML document needs a basic structure:
    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Webpage</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is a paragraph of text.</p>
    </body>
    </html>
    ```
    *   `<!DOCTYPE html>`:  Tells the browser this is an HTML5 document.
    *   `<html>`:  The root element of the page.
    *   `<head>`: Contains information *about* the page, like the title.
    *   `<body>`:  Contains the *visible* content of the page.

!!! Semantic HTML: Giving Meaning to Your Content

*   **What is Semantic HTML?** It's about using HTML tags to describe the *meaning* of your content, not just how it looks.
*   **Why use Semantic HTML?**
    *   **Accessibility:** Helps screen readers understand your content better, making your site usable for everyone.
    *   **SEO (Search Engine Optimization):** Search engines like Google can understand your content better, improving your search ranking.
    *   **Maintainability:** Makes your code easier to read and understand.
*   **Examples of Semantic Elements:**
    *   `<article>`: Represents a self-contained composition in a document, page, application, or site (e.g., a blog post, a news article). Think of it as an independent piece of content that makes sense on its own.
    *   `<nav>`: Represents a section of a page that links to other pages or parts within the page. This is your main menu or navigation bar.
    *   `<footer>`:  Represents the footer for a document or section. Typically contains information about the author, copyright information, or related links.

!!! Lists and Links: Organizing and Connecting Your Content

*   **Unordered Lists (`<ul>`):** For creating lists where the order doesn't matter (like a shopping list). Each item is marked with a bullet point.
    ```html
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>
    ```
*   **Ordered Lists (`<ol>`):** For creating lists where the order is important (like instructions). Each item is numbered.
    ```html
    <ol>
        <li>First step</li>
        <li>Second step</li>
        <li>Third step</li>
    </ol>
    ```
*   **Hyperlinks (`<a>`):** Used to create links to other webpages or resources. The `href` attribute specifies the URL.
    ```html
    <a href="https://www.google.com">Visit Google</a>
    ```
    Analogy: Think of a hyperlink as a portal to another world (website).

!!! Images and Forms: Adding Visuals and Interactivity

*   **Images (`<img>`):** Used to display images on your webpage. The `src` attribute specifies the path to the image file.  The `alt` attribute provides alternative text for the image, important for accessibility (if the image can't load, the text will show).
    ```html
    <img src="myimage.jpg" alt="A beautiful sunset">
    ```
*   **Forms (`<form>`):** Used to collect information from users. Forms contain various input elements.
*   **Form Elements:**
    *   `<input type="text">`: A single-line text input field.
    *   `<input type="email">`:  A single-line email input field.  Browsers often validate the email format.
    *   `<textarea>`: A multi-line text input field.
    *   `<button type="submit">`:  A button that submits the form data.
    *   `<label>`: Associates a text label with an input element, improving accessibility.

!!! Putting it all Together: A Simple Webpage

*   **Example HTML code:**
    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <title>My Simple Webpage</title>
    </head>
    <body>
        <header>
            <h1>Welcome to My Page!</h1>
            <nav>
                <ul>
                    <li><a href="#">Home</a></li>
                    <li><a href="#">About</a></li>
                    <li><a href="#">Contact</a></li>
                </ul>
            </nav>
        </header>

        <article>
            <h2>My First Blog Post</h2>
            <p>This is my first blog post using HTML!</p>
            <img src="placeholder.jpg" alt="Placeholder Image">
        </article>

        <form>
            <label for="name">Name:</label><br>
            <input type="text" id="name" name="name"><br><br>
            <label for="email">Email:</label><br>
            <input type="email" id="email" name="email"><br><br>
            <button type="submit">Submit</button>
        </form>

        <footer>
            <p>&copy; 2023 My Website</p>
        </footer>
    </body>
    </html>
    ```
*   **Explanation:** This code combines headings, navigation, an article, an image, a form, and a footer to create a basic webpage structure.  Experiment with this code and try changing it!  Replace `placeholder.jpg` with an actual image file (or a link to one) to see it in action.
